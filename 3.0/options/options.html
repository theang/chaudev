Please set media.setsinkid.enabled option to true in the Firefox Configuration, to do that:
  • Open 'about:config': type 'about:config' in the address field. And press 'Enter'
  • Click 'I accept the risk!' button to proceeed.
  • Type or copy 'media.setsinkid.enabled' to the search field at the top.
  • Double click 'media.setsinkid.enabled' to change its value to 'true'. Alternatively right-click and select the Toggle command.
  • Value of the 'media.setsinkid.enabled' parameter should be displayed as true
Link to more detailed document on how to access Configuration Editor is here.
